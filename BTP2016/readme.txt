1. Download .jar file from link
2. open eclipse
3. Create a java project named, e.g., ProjectName
4. Right click on the project and select import.
5. Choose General/"Archive File"
6. Browse to the Download Location
7. Select the jar file which you downloaded
8. Click finish.
9. It asks for "Overwrite '.classpath' in folder 'ProjectName'? Click "Yes to All". 
10. To run the application right click on the ProjectName and select RunAs/Java Application/MainClassName-PackageName
11.You can see the output in the console
